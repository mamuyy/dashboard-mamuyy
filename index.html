👥 KPI per PIC

KPI final hanya untuk pegawai organik. Output anak magang/support dihitung ke PIC organik penanggung jawab.
KPI ini adalah alat monitoring berbasis data hasil, proses, dan kualitas. Nilai digunakan untuk melihat posisi, gap, dan prioritas perbaikan. Jika data masih DUMMY/PARTIAL, nilai belum digunakan sebagai penilaian final individu.
Skor adil berbasis hasil + proses + kualitas. KPI ORGANIK/FINAL saat ini memakai PROXY sementara berbasis agregasi skor per PIC accountable (bukan rata-rata akhir anak magang saja) sambil menunggu data mentah owner final.

🧭 Transparansi KPI

Excellent: ≥ 82Good: 70 - 81.99Watch: 58 - 69.99Critical: < 58
Cara Mengejar:
  • Collection Result rendah: fokus invoice nominal besar, overdue tinggi, dan janji bayar lewat jatuh tempo.
  • Follow Up Discipline rendah: update log follow up harian.
  • Pipeline Movement rendah: dorong pergerakan status ND/MIR7/Payment.
  • Data Quality rendah: lengkapi nomor dokumen, status, tanggal, dan PIC.
  • SLA Speed rendah (Tim Surat): selesaikan dokumen paling lama tertahan.
  • Backlog Control rendah: kurangi antrean dokumen belum selesai.
  • Update Discipline rendah: wajib update status harian.
